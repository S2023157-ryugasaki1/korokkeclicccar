# コロッケクリッカー
https://s2023157-ryugasaki1.github.io/korokkeclicccar/
## 概要
茨城県龍ケ崎市ではコロッケを用いて町おこしを行っている。
しかし、龍ケ崎のコロッケの知名度が低いため、このゲームを通してコロッケをより多くの人に知ってもらい龍ケ崎の町おこしに貢献することを目的としたゲームである。

## 動作環境
本ゲームはPC向けに開発されたものです。
Chrome以外での動作は保証されておりません。
